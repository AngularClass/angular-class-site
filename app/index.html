What

Angular Class is a 2 day deep dive into angular created for those who want to the most out of the framework. We created Angular Class because we noticed that there is a lot of want out there for angular but the path to learning can be some what challenging. So we created what we believe will be the best way for yo to learn Angular and start building with it

How you will learn

Rebuild popular apps

From our experience with teaching hundreds of engineers, we've learned that there is nothing more gratifying then rebuilding something that is already there.

Pair Programing

Having someone to learn with will only get you there faster. Pair up with one of your fellow classmates and become an ngMaster

Hands on guidance

Our mentors and instructors have taught hundreds of engineers. They'll be just an arms length away if you feel like you're drowning.

Concepts covered

Directives

Directives are the core feature in Angular. Learn how to use the the built in ones and how to create your own.

Routing and Templates

Angular shines as a single page application (SPA) framework. Like all SPA's, there needs to be a sophisticated client routing feature. Learn how to use core and 3rd party routers.

Controllers

Learn the how to setup logic for your views the Angular way by using controllers

Server interaction

Learn how to use ajax with $http and best practices when communicating with backends.

Advanced Patterns

Angular is very sophisticated, learn how to organize your apps and and write code that won't get you lost.

Testing

Angular shines with testing do to dependency injection and the module system. Learn how to get started with Karma and write unit test
