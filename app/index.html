What

Angular Class is a 2 day deep dive into angular using ES6. We created Angular Class to help engineers level-up and learn how to build awesome things with angular.

How you'll learn

Deep walkthroughs

We cover the core concepts of angular. We'll uncover the magic behind angular so you can start building.

Pair Programing

Pair programming is awesome, especially when learning new things. Pair up with one of your fellow classmates and become an ngMaster.

Hands on guidance

Our mentors and instructors have taught hundreds of engineers. We'll be around to get you un-stuck when you're neck deep in angular.

Concepts covered

Directives

Directives are the core feature in Angular. Learn how to use the built in droectives and also how to create your own.

Routing and Templates

Angular shines as a single page application (SPA) framework. Learn how to use core and 3rd party routers in angular for client side routing.

Controllers

Controllers store all the logic for our views. Learn how to use them correctly and build awesome UI's.

Server interaction

Learn how to use ajax with $http.

Advanced Patterns

Angular is very sophisticated, learn how to organize your apps and and write code that won't rive you and your team crazy.

Testing

Angular shines with testing due to dependency injection and the module system. Learn how to get started with Karma and write unit test.
