Angular Class is a 2 day deep dive into angular using ES6 and WebPack using the component approach.

How you'll learn

Deep walkthroughs

We start with the core concepts of angular and lead into the more advanced topics. We'll uncover the magic so you can start building.

Learn by doing

Along with our lectures into the many parts of angular, you'll be building real world examples and have an app finished on day two.

Hands on guidance

Our mentors and instructors have taught hundreds of engineers. We'll be around to get you un-stuck when you're neck deep in angular.

Concepts covered

Directives

Directives are the core feature in angular. Learn how to use the built in directives and also how to create your own.

Routing and Templates

Angular shines as a single page application (SPA) framework. Learn how to use core and 3rd party routers for client side routing.

Controllers

Controllers store all the logic for our views. Learn how to use them correctly and build awesome UI's.

Server interaction

Learn how to properly handle network I/O with $http in angular.

Advanced Patterns

Angular is very sophisticated, learn how to organize your apps and and write code that won't rive you and your team crazy.

Testing

Angular shines with testing due to dependency injection and the module system. Learn how to get started with Karma and write unit test.
